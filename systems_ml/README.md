# ML Systems Technical Discussion

For every design question, answer in this order:

1. Clarify product goal, traffic, latency, quality, cost, and safety constraints.
2. State the offline and online data path.
3. Choose a baseline architecture before optimizing.
4. Identify the main bottleneck: compute, memory, network, storage, or human labeling.
5. Define metrics, evaluation, rollout, monitoring, and rollback.
6. Name the trade-off you accepted and why.

This track is tuned for applied-science, research-engineering, NVIDIA performance, Amazon AI infrastructure, multimodal/robotics, and quant-system interviews.
